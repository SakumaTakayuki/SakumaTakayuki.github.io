こんにちは、佐久間 崇征です。

## 🔹 個人辞書（単語・用語管理）Webアプリ  
**Python / Streamlit / SQLAlchemy / PostgreSQL（Supabase）**

業務・学習中に出てきた単語や用語を  
**クラウドDBに登録・検索・編集・削除できる個人向け辞書管理ツール**です。

- Streamlit による軽量 UI
- Supabase（PostgreSQL）による永続データ管理
- `user_id` による **ユーザー単位のデータ分離**
- 公開デモ運用を想定した **デモデータ定期削除（Scheduler）**
- DB接続設定・初期化処理の責務分離を意識した構成

実運用・公開を前提とし、  
**セキュリティ・運用面まで含めた設計**を重視しています。

👉 **[プロジェクト詳細を見る](MyDictionary/)**

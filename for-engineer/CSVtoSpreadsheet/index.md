# 📊 売上集計 & 支払報酬 自動計算ツール
**Google スプレッドシート / QUERY / FILTER / ARRAYFORMULA**

CSV を貼り付けるだけで下記を自動化：

- 代理店ごとの売上抽出<br>
- 報酬率に応じた支払額計算<br>
- レポート自動生成<br>
- UI最適化（色・罫線・保護設定）

# ✨ 主な機能
### 1️⃣ 全体売上の自動集計
- 代理店ごとに売上を自動集計<br>

### 2️⃣ 報酬額の自動計算
- 代理店ごとのパーセンテージに応じて報酬額を算出<br>

### 3️⃣ レポート生成（UI付き）
- 代理店名を入力すると売上明細と合計が自動表示<br>
- 存在しない代理店は「代理店が存在しません」と表示<br>
- 見やすい UI（色・罫線・保護範囲）

---

# 🛠 使用関数・技術
- QUERY 関数<br>
- FILTER 関数<br>
- IF・COUNTIF<br>
- ARRAYFORMULA<br>
- 保護範囲設定<br>
- UI デザイン（罫線・色・配置）
